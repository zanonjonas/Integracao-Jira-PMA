Passos para a requisição de um novo token no Jira

  1. Invoque o método de requisição de um token temporário: {ip-atual}/pax/api/v1/processamento/auth/temp_token
  2. Authorize o token temporário no link recebido pelo método
  3. Invoque o método de requisição do token de acesso, passando como parâmetro o código de verificação do jira: {ip-atual}/pax/api/v1/processamento/auth/access_token/{codigo-verificação}
  4. Processe os chamados como ninguem! (Obs: após a execução desses passos, o token será válido até que o mesmo tenha acesso revogado)
